  • Use CDN - it's more flexible when you move things around
  • BE CAREFUL with event listeners. If you delete the html element, it can cause all the JS to stop working
  • NOW I'm starting to udnerstand the importance of separation of concerns
  • There are some great tools for styling sliders - too bad my CSS is not working yet to show the cool slider styling!
